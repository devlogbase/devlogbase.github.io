## [C++] 특수문자 출력하기

<br/>

### 문제설명

    다음과 같이 출력하도록 코드를 작성해 주세요.

<br/>

### 출력예시

    !@#$%^&*(\'"<>?:;

<br/>

### 풀이

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    cin >> str;
    
    int len = str.length();
    
    for (int i = 0; i < len; i++)
    {
        if (str[i] >= 65 && str[i] <= 90)
        {
            str[i] += 32;
        }
        else if (str[i] >= 97 && str[i] <= 122)
        {
            str[i] -= 32;
        }
    }
    
    cout << str;
    
    return 0;
}
```
