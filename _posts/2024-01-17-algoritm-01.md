## [C++] 문자열 출력하기 

<br/>

### 문제

    문자열 str이 주어질 때, str을 출력하는 코드를 작성해 보세요.

<br/>

### 풀이

```cpp
#include <iostream>
#include <string>

using namespace std;

int main(void) {
    string str;
    
    cin >> str;
    cout << str;
    
    return 0;
}
```
